# Aditya Solanki's Portfolio
Welcome to the source code for Shauryam Singh's Portfolio, showcasing a curated collection of projects demonstrating expertise in front-end and back-end development, UI/UX design, and innovative problem-solving.

## 📂 Project Highlights
- Task Manager App — React, HTML, Tailwind CSS
A productivity app to organize and manage tasks efficiently.

- Smart Canteen — Python, MySQL, Flask, Bootstrap
A backend-driven canteen management system with database integration.

- Advance Attendance — Python, Flask, HTML, CSS
An attendance marking system with advanced features and user interface.

- Women Safety — Python, Flask, OpenCV, Mediapipe
Safety app utilizing computer vision and real-time detection.

- Herbal Garden — Python, Flask, OpenCV, Mediapipe
Virtual herbal garden visualization using image processing techniques.

## 🛠 Tech Stack
- Frontend: React, Tailwind CSS, Framer Motion

- Tools: React Slick (carousel), Vercel (deployment)

## ⚙️ Features
- Responsive layout that adapts to all screen sizes

- Animated project cards with hover effects and smooth transitions

- Interactive carousel slider showcasing multiple projects

- External links to live demos and GitHub repositories

- Clean and modern UI design
